Sentencias en JavaScript

Terminan en punto y coma

Es recomendable utilizar ; al final de cada sentencia

Espacios en blanco

Los multiples espacios en blanco se ignoran

Se recomienda utilizar espacios entre operadores =, +, - , /

Lineas de mas de 86 caracteres

Es recomendable dividir la linea despues de un operador

Bloques de codigo

Se agrupan en llaves

Se utilizan en funciones, condiciones, etc

Palabras reservadas

  • break
  • continue
  • switch
  • for
  • do while
  • if else
  • debugger
  • try catch
  • var
  • function
  • return
